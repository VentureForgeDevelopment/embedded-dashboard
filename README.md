# WebLinguist Embedded Dashboard - Public Source Code

This repository contains the public, open-source code for the WebLinguist embedded dashboard application.

## Purpose

This repository exists to provide transparency and to comply with the licensing requirements of platforms where our application is embedded, such as the WordPress.org plugin repository (GPL). The code here is a mirror of our internal, private development repository.

This dashboard is a Vue.js 3 application built with Vite. It is designed to run inside an `<iframe>` or a webview within host applications like WordPress and Shopify, providing a seamless user experience for managing the WebLinguist service.

## Development

To run this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
